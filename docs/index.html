// ABOUT

Alien Invaders is a faithful reimagining of the iconic 1978 Space Invaders arcade classic, rebuilt from scratch using the Godot 4 Engine. Originally developed as a computer-science class presentation, the project demonstrates core game-development concepts — collision detection, wave spawning, score tracking, and player lives — all within a clean, readable GDScript codebase.

// FEATURES

  • Classic grid formation — alien army marches left-right and descends each wave
  • Mystery UFO streaks across the top for bonus points
  • Progressive difficulty — aliens speed up as their numbers dwindle
  • Bunker shields that degrade from both enemy and player fire
  • Persistent high-score tracking per session
  • Authentic retro sound FX and pixel-art sprites
  • Infinite wave loop — how long can you last?
